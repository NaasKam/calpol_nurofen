# CalNur 💊

A simple Progressive Web App (PWA) for creating medication timing reminders.

## Features

- Welcome / terms page
- Editable timing rules
- Calpol timing rule
- Nurofen timing rule
- Combined schedule support
- iOS style time selection
- Checklist
- Dose Taken
- Skip Dose
- History
- Local storage
- iPhone Home Screen support

## Important

CalNur is not a medical device and does not provide medical advice.

Always read medication instructions and consult a healthcare professional.

## Deployment

Upload all files to GitHub Pages.

Settings → Pages → Deploy from main branch.
